              Andy Bargh Mod Akram • 10 months ago

              Akarm, Take a look at the following page on Apple's website. They indicate that Snow Leopard is currently the minimum requirement but your Macbook does need to have an Intel processor. In addition to this, you will also need to be a member of their developer program if you want to run any of your apps on a real device (rather than using the simulator). If you satisfy all these requirements you should ready to go!

              Take a look at the following tutorials when you're ready to get started:
              How to Install Xcode

                        Andy Bargh Mod Jay Kaushal • a year ago

                        Hi Jay, The specs for the MacBook look a good choice for getting started. If you can I might think about adding a bit more memory in it though (I think 8GB is probably the way to go). This is usually cheaper if you do it post-purchase as memory prices from Apple are usually quite expensive compared to other places. If you take this route you will need to do some tinkering to install it yourself though so make your choice depending on how comfortable you are with that sort of thing.

                        Also, I'd make sure that you can hook it up to that external monitor. 13inch screen is great for mobility but does have its down sides and as I said in the article, Xcode loves screen real estate so the bigger screen you can get the better. Be careful with this though. As far as I know, that machine you've indicated only has thunderbolt ports and depending on the spec of your monitor you might need to get a thunderbolt to DVI adapter. Apple sells one but you can also get them from other outlets slightly cheaper. Either way, I've a feeling your son will love you for hooking up the bigger screen.
